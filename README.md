📊 Customer Churn Analysis Project

🔍 Overview

This project focuses on analyzing customer churn behavior using multiple tools — SQL, Excel, and Tableau.
The goal is to identify patterns behind customer churn and derive insights that can help businesses improve retention.


🧰 Tools & Technologies Used

🗄️ SQL (MySQL Workbench)
	•	Performed data cleaning and preprocessing
	•	Created calculated columns such as:
	•	Risk Level (High, Medium, Low)
	•	Customer Segmentation
	•	Used:
	•	CASE statements
	•	Aggregations (COUNT, SUM, AVG)
	•	Window functions (RANK)
	•	Extracted insights like churn rate by contract and customer behavior patterns


📈 Excel Dashboard
	•	Imported cleaned dataset from SQL
	•	Created Pivot Tables for analysis:
	•	Churn by Contract
	•	Tenure vs Churn
	•	Revenue analysis
	•	Built an interactive dashboard using:
	•	Charts (Bar, Stacked Bar, Column)
	•	Filters and slicers
	•	Focused on clear visual storytelling and quick insights


📊 Tableau Dashboard
	•	Developed a fully interactive dashboard
	•	Created calculated fields:
	•	Churn Flag
	•	Tenure Group
	•	Customer Segment
	•	Built multiple visualizations:
	•	Churn by Contract
	•	Churn by Tenure
	•	Revenue by Churn
	•	Risk Segmentation
	•	Customer Segment Analysis
	•	Added filters:
	•	Contract
	•	Gender
	•	Risk Level
	•	Enabled user interaction for dynamic analysis


📌 Key Insights
	•	Customers with month-to-month contracts have the highest churn
	•	Short tenure customers are more likely to leave
	•	High-value customers contribute significantly to revenue loss when churned
	•	Risk segmentation helps identify potential churn groups
